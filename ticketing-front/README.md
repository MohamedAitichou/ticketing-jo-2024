# 💻 Front-end – Ticketing (React + Vite)

## ⚙️ Description
Interface utilisateur développée avec **React + Vite**.  
Elle permet à l’utilisateur :
- de s’inscrire et se connecter (avec OTP),
- d’acheter des billets,
- de visualiser ses commandes.

## 🚀 Lancer le projet

```bash
npm install
npm run dev
